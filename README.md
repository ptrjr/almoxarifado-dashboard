# 📦 Almoxarifado Dashboard

Sistema fullstack para controle de estoque, desenvolvido a partir de problemas reais enfrentados no ambiente de trabalho.

---

## 🚀 Sobre o projeto

Entrei recentemente na área de desenvolvimento e decidi aprender na prática, usando dificuldades do meu trabalho atual como base para construir soluções reais.

Este é meu primeiro projeto fullstack. Mesmo com pouco tempo de estudo, consegui desenvolver um sistema funcional — e estou satisfeito com a evolução até aqui, sabendo que ainda há muito a melhorar.

---

## 🎯 Problema que o projeto resolve

Na rotina da empresa, existe dificuldade em manter um controle claro e confiável do estoque.

Este sistema foi criado para:

- Organizar entradas e saídas de materiais  
- Melhorar a visibilidade do estoque  
- Reduzir erros operacionais  
- Facilitar o acompanhamento de movimentações  

⚠️ Importante:  
Este projeto melhora o controle de estoque, mas **não resolve problemas estruturais de produção**, como perdas por corte ou sobras de materiais.

Esses desafios serão tratados em um projeto futuro chamado **Arcos**, com foco em cálculo e rastreamento de perdas na produção.
